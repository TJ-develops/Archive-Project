# The Archive — Complete Site Flow

---

## Overall Flow

```
LANDING PAGE → MAIN HUB → SECTIONS → PROFILES/PAGES
```

---

## Page by Page Flow

### 1. Landing Page
```
User arrives
Sees site name, tagline, clean design
Clicks "Enter the Archive"
Goes to Main Hub
```

---

### 2. Main Hub
```
User sees all major sections laid out
Can click any section → goes to that section
OR
Uses global search bar to jump straight to content
```

**Sections visible on Main Hub:**
- Case Files
- Key Figures
- Organizations
- Operations
- Timeline
- Documents Vault
- Whistleblowers
- Rabbit Holes

---

### 3. Case Files Section
```
User enters
Sees archive room with A-Z shelves
Browses shelves
Finds a case box
Clicks it
Opens Case File Page
    │
    ├── Overview Tab
    ├── Timeline Tab
    ├── Key Players Tab ──────────→ links to Key Figures profiles
    ├── Evidence Tab
    ├── Documents Tab ────────────→ links to Documents Vault
    └── Impact Tab
              │
              └── Related Cases ─→ links to other Case Files
```

---

### 4. Key Figures Section
```
User enters
Sees filter options:
    ├── Hit List
    ├── Still At Large
    ├── Red Notice
    ├── Convicted
    ├── Pardoned
    ├── Deceased
    ├── Disappeared
    └── Untouchable

User picks a filter
Sees grid of profile cards
Clicks a profile
Opens Figure Profile Page
    │
    ├── Status Badge
    ├── Personal Info
    ├── Wanted For
    ├── Connected Cases ──────────→ links to Case Files
    ├── Connected Organizations ──→ links to Organizations
    ├── Known Associates
    ├── Timeline of Involvement
    └── Current Status / Last Known Location
```

---

### 5. Organizations Section
```
User enters
Sees filter by type:
    ├── Government Agencies
    ├── Corporations
    ├── Intelligence Agencies
    ├── Military
    ├── Cults
    └── Criminal Organizations

User picks type
Sees organization cards
Clicks one
Opens Organization Profile
    │
    ├── Overview & History
    ├── Structure & Leadership
    ├── Connected Cases ──────────→ links to Case Files
    ├── Key Figures involved ─────→ links to Key Figures
    ├── Operations run ───────────→ links to Operations
    ├── Funding Sources
    └── Current Status
```

---

### 6. Operations Section
```
User enters
Sees all named operations (MKUltra, Paperclip, COINTELPRO etc.)
Can filter by:
    ├── Intelligence
    ├── Military
    ├── Medical
    ├── Financial
    └── Foreign

Clicks an operation
Opens Operation Profile
    │
    ├── Overview
    ├── Timeline
    ├── Objectives (Official vs Real)
    ├── Key People ───────────────→ links to Key Figures
    ├── Documents ────────────────→ links to Documents Vault
    └── Impact
```

---

### 7. Timeline Section
```
User enters
Sees full historical timeline
All cases, events, operations plotted chronologically
Can filter by:
    ├── Type (political, corporate, military)
    ├── Country
    └── Status

Clicks any point on timeline
Goes directly to that Case / Operation / Figure page
```

---

### 8. Documents Vault
```
User enters
Sees all primary source documents across entire archive
Filter by:
    ├── Declassified Government Docs
    ├── Leaked Files
    ├── Court Records
    ├── FOIA Releases
    └── Reports & Investigations

Clicks a document
Opens document viewer
    │
    └── Links back to related Case / Operation
```

---

### 9. Whistleblowers Section
```
User enters
Sees profiles of people who exposed wrongdoing
Organized by:
    ├── Government
    ├── Corporate
    ├── Military
    └── Intelligence

Clicks a profile
Opens Whistleblower Profile
    │
    ├── Who They Are
    ├── What They Exposed
    ├── What Happened to Them
    ├── Connected Cases ──────────→ links to Case Files
    └── Documents They Leaked ────→ links to Documents Vault
```

---

### 10. Rabbit Holes Section
```
User enters
Sees curated investigative journeys

Example path: "Follow the CIA Thread"
    MKUltra
        ↓
    Project Paperclip
        ↓
    Operation Gladio
        ↓
    P2 Lodge
        ↓
    Vatican Bank Scandal

User follows the path
Each stop is a full Case / Operation page
```

---

## How Everything Connects

```
Case File      ←→   Key Figures
Case File      ←→   Organizations
Case File      ←→   Operations
Case File      ←→   Documents Vault
Key Figures    ←→   Organizations
Key Figures    ←→   Operations
Organizations  ←→   Operations
Timeline            links to everything
Documents Vault     links to everything
Rabbit Holes        weaves through everything
```
